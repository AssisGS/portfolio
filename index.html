Conhecimento

Ambientes Computacionais

  • Conjunto de recursos e ferramentas para desenvolvimento e execução de software.
  • Inclui hardware, sistemas operacionais, servidores e redes.
  • Tipos comuns: local (PC), remoto (servidores) e virtualizado (nuvem).
  • Exemplo: ambientes de desenvolvimento integrado (IDEs), como Visual Studio e Eclipse.

Lógica de Programação

  • Base do raciocínio para criar algoritmos e resolver problemas.
  • Envolve a criação de fluxos de decisão, repetição e interação.
  • Usa conceitos como variáveis, operadores, estruturas condicionais (if, else) e loops (for, while).
  • Fundamental para entender qualquer linguagem de programação.

Front-end

  • Parte visual de um site ou aplicativo que o usuário interage.
  • Envolve HTML, CSS e JavaScript para criar interfaces atraentes e funcionais.
  • Foco na usabilidade e experiência do usuário (UX).
  • Frameworks populares: React, Angular e Vue.js.

Banco de Dados

  • Sistema que armazena e organiza grandes volumes de dados.
  • Permite a criação, leitura, atualização e exclusão de dados (CRUD).
  • Tipos: Relacional (SQL) e Não-relacional (NoSQL).
  • Exemplo de sistemas de gerenciamento: MySQL, PostgreSQL, MongoDB.
